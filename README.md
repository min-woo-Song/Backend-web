# board & login
* 스프링 시큐리티를 이용한 로그인, 로그아웃 구현
* Google, Naver, Kakao OAuth2 login 구현
* 게시판 CRUD, 검색 기능, 페이징 구현
* 게시글 작성자에 해당하는 계정만 수정,삭제가 가능

# 기술스택
* Java 11
* Thymeleaf
* Spring Boot
* Spring Security
* MySQL
* MyBatis
* OAuth2.0
